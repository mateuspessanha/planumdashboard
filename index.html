Planum Solution
Configure o Supabase para sincronizar dados em tempo real entre todos os dispositivos.
1
Crie conta gratuita em supabase.com → New Project
2
SQL Editor → cole e execute:
create table sellers ( id text primary key, name text not null, color text not null, team text not null default 'comercial', created_at timestamptz default now() ); create table sales ( id text primary key, seller_id text references sellers(id) on delete cascade, date text not null, amount numeric not null, updated_at timestamptz default now(), unique(seller_id, date) ); create table settings (key text primary key, value text not null); alter table sellers enable row level security; alter table sales enable row level security; alter table settings enable row level security; create policy "pub sellers" on sellers for all to anon using (true) with check (true); create policy "pub sales" on sales for all to anon using (true) with check (true); create policy "pub settings" on settings for all to anon using (true) with check (true);
3
Settings → API Keys → aba Legacy → copie URL e chave anon
